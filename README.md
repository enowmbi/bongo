# Bongo
Rails engine for blogging using MongoDB.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'bongo'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install bongo
```
